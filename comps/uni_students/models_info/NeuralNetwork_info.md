# Neural Network (MLPClassifier)

Generated by `comps/uni_students/scripts/run_extended_analysis.py` when the Kaggle train/test CSVs are available locally.
This model is optional and only trained when `--include-mlp` is passed.

## Intended role
- Neural baseline for the cleaned tabular feature set
- Uses `StandardScaler` inside a sklearn pipeline before the MLP

## Validation setup
- 5-fold stratified cross-validation
- Early stopping enabled during training
- Compact grid search over hidden-layer sizes, learning-rate initialization, and L2 regularization

## Output artifacts
- `comps/uni_students/submissions/submission_mlp.csv`
- Updated cross-validation summary and out-of-fold classification report in this file
