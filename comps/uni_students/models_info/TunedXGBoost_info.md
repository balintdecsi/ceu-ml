# Tuned XGBoost

Generated by `comps/uni_students/scripts/run_extended_analysis.py` when the Kaggle train/test CSVs are available locally.

## Intended role
- Strong tree-based baseline for the cleaned tabular features
- Uses unscaled features so one-hot flags and missing indicators keep their natural interpretation

## Validation setup
- 5-fold stratified cross-validation
- Compact grid search over depth, learning rate, number of trees, and child-weight regularization
- Bootstrap confidence interval computed from the fold accuracies

## Output artifacts
- `comps/uni_students/submissions/submission_xgb_tuned.csv`
- Updated cross-validation summary and out-of-fold classification report in this file
